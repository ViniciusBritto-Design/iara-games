# Iara Games

## 1. Contexto do projeto

A Iara Games é um projeto acadêmico desenvolvido para a Sprint 01 da disciplina de Web Design. Nesta etapa inicial, o objetivo é construir a visão conceitual e visual da plataforma, além de desenvolver uma página inicial estática em HTML e CSS e organizar o projeto em um repositório público no GitHub.

## 2. Proposta da plataforma

A Iara Games é uma plataforma digital de distribuição de jogos independentes, com foco na valorização de desenvolvedores emergentes, especialmente brasileiros, e na oferta de uma experiência acessível, intuitiva e visualmente atrativa para diferentes perfis de jogadores.

A proposta da plataforma é conectar criadores e jogadores em um ambiente digital organizado, moderno e inclusivo, servindo como base para futuras expansões do projeto nas próximas sprints.

## 3. Pesquisa de referências

Para orientar as decisões iniciais do projeto, foram analisadas três plataformas de distribuição de games como referência:

### Steam
Referência importante pela amplitude do catálogo, estrutura de navegação e organização de conteúdos. Foi observada como exemplo de plataforma robusta, embora apresente interface mais densa.

### Epic Games Store
Utilizada como referência pela abordagem mais limpa e visualmente organizada, com maior destaque para banners, jogos em evidência e hierarquia visual clara.

### itch.io
Importante por seu foco em jogos independentes e liberdade visual. Serviu como referência conceitual para entender como uma plataforma voltada a jogos indie pode valorizar identidade e criatividade.

## 4. Decisões de design

### Paleta de cores
- Verde escuro (#0F3D3E)
- Verde claro (#2EC4B6)
- Branco (#FFFFFF)
- Azul escuro / grafite (#111827)

### Justificativa da paleta
A paleta foi definida com predominância de tons de verde para reforçar a identidade da marca Iara, associando o projeto a elementos naturais e à ideia de originalidade. Os tons escuros ajudam a criar contraste, conforto visual e aproximação com interfaces comuns no universo gamer.

### Tipografia
- Poppins: utilizada em títulos e elementos de destaque
- Roboto: utilizada em textos e conteúdos gerais

### Justificativa tipográfica
A combinação entre Poppins e Roboto foi escolhida por unir modernidade, clareza e boa legibilidade, contribuindo para uma interface mais organizada e acessível.

### Decisões visuais adotadas
1. Uso de layout com destaque para seções principais e jogos em cards.
2. Aplicação de tema escuro para melhorar conforto visual e coerência com o segmento de games.
3. Organização da informação com hierarquia visual clara entre títulos, textos, botões e áreas de navegação.

## 5. UX e acessibilidade

### Cuidados de usabilidade
Foram adotados princípios de usabilidade com foco em:
- navegação simples e objetiva;
- organização previsível dos conteúdos;
- destaque para ações principais;
- estrutura visual escaneável e de fácil leitura.

### Cuidados de acessibilidade
Foram considerados os seguintes aspectos:
- contraste adequado entre fundo e texto;
- uso de estrutura semântica em HTML;
- navegação com links claros;
- tipografia legível em diferentes tamanhos de tela;
- organização responsiva para melhor adaptação a dispositivos diversos.
